Why Should I Use This?
************************************************************

This is a Python package for data analysis which contains some very useful functions. 


Installation
************************************************************

$ pip install dplearn



Modules
************************************************************

- tools
- math



Usage
************************************************************

from dplearn.tools import clean
data = clean(data)
print(data)
